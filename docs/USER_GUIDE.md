# MedRoundTable 用户快速入门指南

## 🎯 5分钟快速上手

### 第一步：登录平台
1. 访问 MedRoundTable 网站
2. 点击 "使用 SecondMe 登录"
3. 授权 Second Me 账号访问
4. 选择你的专业角色

### 第二步：发起圆桌讨论
1. 点击 "新建圆桌会"
2. 输入临床问题（例如："二甲双胍对2型糖尿病患者心血管结局的影响"）
3. 选择参与的 AI Agent
4. 点击 "开始讨论"

### 第三步：与AI协作
- **临床主任**：会评估研究价值和可行性
- **博士生**：会进行文献检索和背景调研
- **流行病学家**：会设计研究方案
- **统计专家**：会制定分析计划
- **研究护士**：会设计数据收集流程

### 第四步：获取成果
讨论完成后，平台自动生成：
- 📋 研究方案文档
- 📊 CRF数据采集表
- 📈 统计分析计划
- 📖 操作手册

## 🤖 5个AI Agent介绍

### 1. 👨‍⚕️ 资深临床主任
**专业领域**：临床医学、科研价值评估
**主要职责**：
- 评估临床问题的科研价值
- 判断研究假设的可行性
- 指导研究方向
- 审核最终成果

**使用场景**：
- 不确定某个临床现象是否值得研究
- 需要专家判断研究设计的临床意义
- 评估研究结果的临床应用价值

### 2. 📚 临床博士生
**专业领域**：文献检索、学术写作、项目协调
**主要职责**：
- 进行系统性文献检索
- 撰写文献综述
- 协调各Agent工作
- 生成研究报告

**使用场景**：
- 需要了解某个研究领域的最新进展
- 撰写研究背景和相关工作
- 整理讨论记录和成果文档

### 3. 📊 临床流行病学家
**专业领域**：研究设计、方法学、质量控制
**主要职责**：
- 设计研究方案
- 制定纳入排除标准
- 评估偏倚风险
- 指导质量控制

**使用场景**：
- 设计随机对照试验
- 制定观察性研究方案
- 评估研究的方法学质量

### 4. 📈 数据统计专家
**专业领域**：统计学、数据分析、可视化
**主要职责**：
- 计算样本量
- 设计CRF表格
- 制定分析计划
- 生成统计图表

**使用场景**：
- 确定研究需要多少样本
- 设计数据采集表格
- 选择合适的统计方法
- 解释统计结果

### 5. 👩‍⚕️ 研究护士
**专业领域**：临床操作、数据采集、质量管理
**主要职责**：
- 设计数据采集流程
- 制定操作规范
- 评估执行难度
- 确保数据质量

**使用场景**：
- 设计患者随访计划
- 制定数据录入规范
- 评估研究的可操作性

## 💡 使用场景示例

### 场景1：糖尿病新药研究
**临床问题**："SGLT2抑制剂对糖尿病肾病进展的影响"

**讨论流程**：
1. 临床主任评估研究价值
2. 博士生检索相关文献
3. 流行病学家设计RCT方案
4. 统计专家计算样本量和终点指标
5. 研究护士设计随访和数据收集流程

**输出成果**：
- 完整的RCT研究方案
- 病例报告表（CRF）
- 统计分析计划
- 研究者操作手册

### 场景2：回顾性队列研究
**临床问题**："某三甲医院5年ICU患者死亡率相关因素分析"

**讨论流程**：
1. 流行病学家设计回顾性研究方案
2. 统计专家确定分析策略
3. 研究护士评估数据可及性
4. 博士生协助伦理审查申请
5. 临床主任审核研究意义

**输出成果**：
- 回顾性研究方案
- 数据提取表格
- 统计分析代码
- 论文框架

### 场景3：临床预测模型
**临床问题**："建立心衰患者30天再入院风险预测模型"

**讨论流程**：
1. 统计专家指导模型选择（Logistic/机器学习）
2. 流行病学家确定预测因子
3. 研究护士评估数据采集可行性
4. 博士生协助文献对标
5. 临床主任验证临床实用性

**输出成果**：
- 模型开发方案
- 变量清单
- 模型验证策略
- 临床实施建议

## 📋 常见问题

### Q1: 圆桌讨论需要多长时间？
**A**: 根据问题复杂度：
- 简单问题：15-30分钟
- 标准研究设计：1-2小时
- 复杂多中心研究：半天-1天

### Q2: 生成的成果可以直接用于伦理申请吗？
**A**: 可以作为基础，但建议：
- 根据实际情况调整
- 请所在机构伦理委员会审核
- 补充机构特定的要求

### Q3: AI Agent的回答可靠吗？
**A**: 
- AI Agent基于大量医学文献训练
- 提供的是参考性建议
- **必须经过专业人士审核**
- 不能替代专业判断

### Q4: 可以导出什么格式的成果？
**A**: 支持多种格式：
- Word文档 (.docx)
- PDF文档
- Excel表格 (.xlsx)
- Markdown文本

### Q5: 如何保证数据隐私？
**A**: 
- 平台不存储患者隐私数据
- 建议输入脱敏后的数据
- 遵守所在机构的隐私政策
- 符合HIPAA/GDPR要求

## 🎓 最佳实践

### 1. 提出好问题
✅ **好的临床问题**：
- "二甲双胍能否延缓2型糖尿病患者的肾功能下降？"
- "ICU患者早期活动对住院时间的影响？"

❌ **需要改进的问题**：
- "糖尿病怎么治？"（太宽泛）
- "这个药有用吗？"（不具体）

### 2. 充分利用AI Agent
- **不要跳过任何Agent**：每个Agent都有独特价值
- **迭代讨论**：可以多次修改问题
- **保存中间结果**：随时可以回溯

### 3. 验证AI建议
- 与同行讨论AI生成的方案
- 查阅最新指南验证建议
- 根据实际情况调整

### 4. 持续学习
- 关注平台更新的医学知识
- 参与社区讨论
- 分享使用经验
