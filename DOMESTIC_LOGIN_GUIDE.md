# 🌏 国内可用登录方案

## 问题背景

原系统使用 **Second Me OAuth** 登录，需要访问国外服务（`https://go.second.me`），在国内没有科学上网的情况下无法使用。

## ✅ 解决方案

我们创建了全新的 **国内直连登录系统**，无需 VPN 即可使用全部功能！

---

## 🔐 三种登录方式

### 1️⃣ 游客模式（推荐）
**无需注册，立即体验**

- ✅ 零门槛使用
- ✅ 全部功能开放
- ✅ 数据保存在浏览器本地
- ✅ 支持创建圆桌、AI 对话、导出报告

**适合场景**：
- 首次体验平台
- 临时使用
- 不想注册账号

---

### 2️⃣ 账号登录
**使用邮箱/手机号 + 密码登录**

**测试账号**：
```
管理员账号：
- 邮箱：admin@medroundtable.com
- 密码：admin123

研究员账号：
- 邮箱：researcher@medroundtable.com
- 密码：research123
```

**适合场景**：
- 需要保存个人设置
- 长期使用平台
- 团队协作

---

### 3️⃣ 注册新账号
**创建自己的专属账号**

需要填写：
- 用户名
- 邮箱
- 密码（至少8位）
- 机构/医院（可选）

**适合场景**：
- 正式使用平台
- 需要个性化设置
- 参与团队项目

---

## 🚀 使用步骤

### 访问新的登录页面
```
https://medroundtable-v2.vercel.app/login-v2.html
```

### 推荐流程
1. **首次访问** → 选择「游客模式」立即体验
2. **喜欢平台** → 注册自己的账号
3. **长期使用** → 用注册的账号登录

---

## 🔒 数据安全说明

### 游客模式
- 所有数据存储在浏览器 `localStorage`
- 清除浏览器数据会丢失
- 单设备使用

### 注册账号
- 数据同样存储在浏览器本地
- 使用 token 验证身份
- 建议定期导出重要数据
